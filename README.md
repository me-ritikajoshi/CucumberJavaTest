
# Cucumber Java Test Automation

BDD-style test automation project using **Cucumber + Java** for readable, business-aligned test scenarios.

## Objective

- Write behavior-focused automated tests using Gherkin
- Improve collaboration between QA and product teams
- Keep acceptance criteria and automation aligned

## Tech Stack

- Java
- Cucumber (Gherkin)
- Maven
- JUnit/TestNG (based on project setup)

## Project Structure

- `src/test` - feature files, step definitions, runners
- `pom.xml` - dependencies and test execution config

## Run Tests

```bash
mvn clean test
